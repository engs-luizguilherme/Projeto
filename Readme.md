# Estufa Inteligente Automatizada

 ### Descrição do Projeto

A Estufa Inteligente Automatizada é um sistema desenvolvido para monitorar e controlar, de forma automática e remota, as condições ambientais de uma estufa agrícola.

O projeto utiliza sensores e atuadores integrados a tecnologias de Internet das Coisas (IoT), permitindo o acompanhamento em tempo real de variáveis como temperatura, umidade do ar e umidade do solo. Com base nesses dados, o sistema realiza ações automáticas, como irrigação, ventilação e controle de iluminação, garantindo melhores condições para o cultivo.

Além disso, o sistema conta com uma interface web interativa (dashboard), onde o usuário pode visualizar dados, receber alertas e realizar ajustes manuais quando necessário.

 ## Objetivo

Automatizar o controle ambiental de uma estufa, aumentando a eficiência da produção agrícola, reduzindo desperdícios e melhorando a qualidade das plantas.


 ## Funcionalidades Principais

* Monitoramento de temperatura
* Monitoramento de umidade do ar e do solo
* Controle automático de irrigação
* Controle de ventilação
* Controle de iluminação
* Sistema de login e autenticação
* Registro e histórico de dados
* Dashboard de monitoramento em tempo real
* Alertas automáticos (temperatura e umidade)
